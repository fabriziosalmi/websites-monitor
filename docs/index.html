‚ö° Quick Start Examples

Single Website Check

# Check a single website with all available tests curl "http://localhost:8000/monitor/single?website=example.com"

Specific Security Check

# Run SSL certificate check only curl "http://localhost:8000/check/ssl_cert?website=example.com"

Batch Monitoring

# Monitor multiple websites curl -X POST "http://localhost:8000/monitor" \ -H "Content-Type: application/json" \ -d '{"websites":["example.com","google.com"]}'

Generate Report

# Generate comprehensive HTML report curl -X POST "http://localhost:8000/generate-report" \ -H "Content-Type: application/json" \ -d '{"websites":["example.com"],"output_format":"html"}'
